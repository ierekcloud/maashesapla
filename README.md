# Maaş Hesaplama Uygulaması

Bu uygulama tek bir HTML dosyası olarak birleştirilmiştir.

## Tek Dosya Sürümü
Uygulamanın çalışmaya hazır, bütün kodları (HTML+CSS+JS) içeren tek dosyasına şu yollardan ulaşabilirsiniz:
1. Ana dizindeki **MaasHesaplama_SON_HALI.html** dosyası.
2. **public/MaasHesaplama_SON_HALI.html** dosyası.

Zip dosyasını şimdi tekrar indirebilirsiniz.
